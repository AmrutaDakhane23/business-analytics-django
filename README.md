# Business Analytics Dashboard (Django + MySQL)

## Description
A Django-based mini project to analyze business records and generate analytics:
- Total Value
- Average Value
- Maximum Value
- Minimum Value

## Tech Stack
- Python
- Django
- MySQL
- HTML/CSS

## Features
- View business records
- Filter by category
- Update values dynamically
- Analytics calculation
